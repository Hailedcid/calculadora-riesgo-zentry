# calculadora-riesgo-zentry
Created with CodeSandbox

Se trata de una calculadora de riesgo para añadir a tu sitio, personalizable en tus colores e información. Esete codigo permite que se añada la funcionalidad a tu sitio y generar leads de forma sencilla y al alcance de todo tipo de cliente, permite recabar datos y poder enviar mensajes.

Una herramienta web interactiva construida en React para evaluar vulnerabilidades de seguridad en inmuebles (Hogar y Empresa).
## 🏢 Sobre el Proyecto
Esta aplicación fue desarrollada específicamente para **Zentry Seguridad**, una empresa con más de 10 años de experiencia en el diseño de sistemas a medida e Ingeniería de Precisión.

La herramienta está diseñada para operar bajo el contexto económico, técnico y normativo de México, realizando cálculos en Pesos Mexicanos (MXN) y sugiriendo implementaciones tecnológicas basadas en los estándares operativos de Zentry Seguridad (CCTV, Alarmas AX PRO, Control de Acceso, etc.)

## 🤖 Colaboración y Autoría (AI-Assisted Development)
**Nota del Autor:** La lógica de negocio, arquitectura final, correcciones de sintaxis, diseño de interfaz y despliegue de esta aplicación fueron realizados íntegramente por mi autoría. Para acelerar el proceso de escritura de código, me apoyé en el uso de modelos de lenguaje de Inteligencia Artificial Avanzada (**Claude** y **Google Gemini**) quienes actuaron estrictamente como asistentes de codificación (Copilots) bajo mis instrucciones y parámetros de Ingeniería de Precisión.

## ⚙️ Características Principales
* **Evaluación B2B / B2C:** Flujos dinámicos dependiendo si el usuario evalúa una casa o un negocio.
* **Motor de Riesgo:** Algoritmo matemático que evalúa medidas actuales vs. vulnerabilidad de la zona[cite: 91, 92].
* **Dashboard Interactivo:** Gráficas en tiempo real generadas con Recharts[cite: 248].
* **Integración CRM (WhatsApp):** Call To Action dinámico que recopila la información del diagnóstico y la envía preformateada directamente al equipo de ingenieros de Zentry[cite: 739].
* **Tech Stack:** React (JSX), CSS puro (Custom Properties), Vercel (Deployment)[cite: 562, 131].
